# *🛰️ Smallsat Space Optics Research*
This repository promotes space-based optical systems for small satellites. It combines models, simulations, analysis tools, and experimental code to investigate how compact optical payloads can be designed, evaluated, and flown in resource-constrained space environments, with the goal of improving the capability, reliability, as well as accessibility of smallsat optics—from early-stage concepts to flight-ready systems. The repository is built to support research iteration, not just final results, and includes but is not limited to
  - Optical system models and simulations
  - Performance analysis (resolution, SNR, alignment, tolerancing, etc.)
  - Supporting scripts, datasets, and experiment results
  - Documentation and references for space optics research
## *How to use*
Each major folder includes a brief explanation of its purpose and expected inputs/outputs for:
 - Clone the repository
